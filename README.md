**普通用户功能：**
1. 登录
2. 注册
3. 宠物商品
4. 宠物服务预约
5. 宠物伴侣服务、寻找伴侣、发布宠物信息
6. 宠物论坛
7. 购物车
8. 我的订单
9. 萌宠视频
10. 个人信息
11. 修改密码
12. 退出登录


**管理员功能：**
1. 登录

4. 宠物伴侣管理

9. 萌宠视频管理

11. 修改密码
12. 安全退出
